# BatApp
A dark, immersive "Bat-OS" mobile app hidden behind a Wayne Manor library. Created by Abhigyan for inTech 2026, this interactive experience seamlessly blends billionaire executive management with vigilante tactical operations.

Core Features

Tactical Radar & Arsenal: Live topographic mapping with automated arsenal drones, Batmobile routing, and EMP override capabilities.

Secure Comms: AI-driven chat for secure communication with Oracle and team.

Wayne Enterprises Hub: A dynamic daily executive agenda synced to a PR assistant that reads, drafts emails, and updates your schedule.

Batcomputer DB: Forensic tools including a clue analyser, GCPD sketch synthesizer, and CCTV threat analysis.

Atmosphere: All set to a soothing but revitalizing Zimmer-esque score.

How to Access & Use

Unlock the System: Open the "Gotham History" book. Go to page 89. Passphrase: GothamWayne.

Deploy Arsenal: Click UAV or EMP buttons to neutralize red radar blips.

Manage Agenda: In the Wayne tab, type "Cancel my 9 PM" to see the AI draft an email and erase the meeting.

Investigate: Upload images to the CCTV scanner or type physical descriptions into the GCPD Sketch generator.
